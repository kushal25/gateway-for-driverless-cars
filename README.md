# Gateway for Driverless Cars

Gateway for driverless cars using LWM2M Protocol following the OMA Specification. LWM2M is a communication protocol that is used between client and a server.

LWM2M Server was developed in Java, Jersey and MongoDB.
LWM2M Client was a simple webapplication developed using HTML, CSS, Javascript and JQuery. The client displayed all data, and if values are modified, it takes appropriate actions based on LWM2M Server responses.
